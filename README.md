# Javascript  
##javascript语法
1.javascript语句向浏览器发出命令，告诉浏览器该做什么  
2.分号:语句之间的分割是分号。可用也可不用。  
3.javascript按照编写顺序依次执行  
4.javascript标识符必须以字母，下划线或美元符号开始。  
5.javascript对大小写敏感  
6.javascript会忽略多余的空格
7.单行注释：//   多行注释：/**/
